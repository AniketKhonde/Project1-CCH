What is CSS?

CSS stands for Cascading Style Sheets. It is a stylesheet language that is used to describe the visual presentation of a web page written in HTML (Hypertext Markup Language).

HTML creates the structure of the page, while CSS adds styling to that structure. This tutorial assumes that you have prior knowledge of HTML. If that's not the case, you can follow the HTML tutorial first.

Why the word "cascade"?

The term "cascade" refers to the priority scheme determining which CSS rules are applied when multiple rules target an element. This scheme takes into account specificity and inheritance, ensuring that the most specific and latest rules are applied.

Why use CSS?

CSS is used to provide styling to HTML elements, making web pages visually appealing and user-friendly.

Analogy to Understand CSS

Imagine reading a book with only plain text. It's quite dull, isn't it? Now picture a book enriched with different colors, fonts, and styles. That's what CSS does to a webpage.

How does CSS work?

CSS operates by selecting HTML elements and applying styles to them. Styles dictate the appearance of elements on a webpage. You can target HTML elements, classes, or IDs, defining properties like colors, fonts, margins, etc.

Syntax of CSS


CSS follows a rule-based structure. Each rule consists of a selector and a declaration block. Selectors pick the HTML elements, while declaration blocks contain pairs of properties and values.

The general syntax for writing CSS:


selector {
    property: value;
}

Note: Semi-colon (;) at the end of each new property and property value is IMPORTANT.

For example:
h2 {
    color: blue;
}
In the example:

  • h2: h2 is the selector.
  • color: It's the property.
  • blue: The property value.

Within the declaration block, there can be multiple pairs of properties and values.
Consider the example:
button {
    color: white;
    background-color: black;
    border: transparent;
    border-radius: 5px;
}
Here, 'button' is the selector, and there are multiple pairs of properties and values. Each pair is separated by a semicolon ";".

Ways to Add CSS

There are three different ways to add CSS to an HTML page, which are:

  • Inline CSS: Inline CSS is used to add custom properties to specific elements. The added style will only reflect on that particular element only.

    • To use inline CSS, Insert the "style" attribute within the HTML element's opening tag.

    <h1 style="color: purple;">I'm Harry</h1> <h2>I'm CodeWithHarry</h2>

    Here, the inline style of color: purple is attached to only the h1 tag.

    Note: The downside of using inline CSS is that once the project complexity increases, it will become difficult to manage the styles of each and individual elements.

  • Internal CSS: Internal CSS is used to apply custom style to multiple elements on the same page. The style can be used throughout the HTML page.

    • Internal CSS is defined in a style block, which will be inside the head section.

    <head>
        <style>
            p {
                color: red;
            }
        </style>
    </head>

    Here, in the style block, selector p will target all p tags and assign them color: red.

    Note: The <style> block should always be in the <head> section.

  • External CSS: External CSS works similarly to internal CSS but with a twist. Instead of adding the styles within the HTML file, we create a separate file with .css extension. This file will hold all the styling details. Then, we link this file to the HTML page, giving it the instructions on how to look.

    • The following video will explain how to link a CSS file to HTML.

    There is a new <link> tag in the head section, and this link tag has rel and href properties.

    The following points will explain each keyword's meaning:

    <link rel="stylesheet" href="style.css">

    Here, <link> tag is used to create links between different resources, like stylesheets, fonts, and more. In our case, we are using a link tag to link the CSS file with the HTML file.

    rel="stylesheet": rel stands for relationship; this defines the type of relationship between the HTML document and the linked resource. When set to "stylesheet", it specifies that the linked resource is a stylesheet that will be used to style the HTML content.

    href="style.css": The href attribute stands for "hypertext reference." It specifies the path or URL to the external resource we want to link. In this case, it's the path to the external CSS file called "style.css".

    <head>

CSS Selectors

CSS selectors allow us to choose specific elements and apply styles to them. Suppose we want to add a custom style to only a specific tag(s). There, we can make use of CSS selector.

There are different types of CSS selectors, which are as follows:

  • Universal Selector
  • Element Selector
  • ID Selector
  • Class Selector
  • Group Selector

Let's look into these selectors one by one

Universal Selector

Universal selector represented by "*" targets all the HTML elements on the page.

The syntax of Universal Selector is as follows:

* {
    property: value;
}

Consider the code snippet:

<html>
<head>
    <style>
        * {
            color: purple;
            text-align: center;
        }
    </style>
</head>

Notice, Irrespective of the tag, the style will be applied to all the elements and tags.

Summary:

  • Universal Selector(*): Target the entire page.
  • Element Selector: Target a specific element.
  • ID Selector(#): Target element with a specific ID.
  • Class Selector(.): Target element(s) with the same class.
  • Group Selector: Group elements and target them.

CSS Backgrounds

The Background Property in CSS

The background: property in CSS is a versatile and powerful property that allows you to set various background-related styles for an element. It can be applied to most HTML elements to control the appearance of their backgrounds.

The background property is shorthand for several individual background-related properties, and it can include values for the following properties:

  • background-color:
    Defines the background color of an element. 
    background-color: #f0f0f0;

  • background-image:
    Specifies the URL of an image to be used as the background. 
    background-image: url('background.jpg');

  • background-repeat:
    Determines how a background image should be repeated. 
    background-repeat: repeat-x;
    Common values include repeat, repeat-x, repeat-y, and no-repeat.

  • background-position:
    Sets the starting position of the background image. 
    background-position: center top;
    Values can be a combination of keywords (center, top, bottom, left, right) or percentages.

  • background-size:
    Specifies the size of the background image. 
    background-size: cover;
    Values can be auto, cover, contain, or specific dimensions.

  • background-attachment:
    Specifies whether the background image should scroll with the content. 
    background-attachment: fixed;
    Values include scroll, fixed, and local.

  • background-origin:
    Sets the background positioning area for the background-image property. 
    background-origin: content-box;
    Values can be padding-box, border-box, or content-box.

  • background-clip:
    Specifies how far the background should extend within an element. 
    background-clip: padding-box;
    Values include border-box, padding-box, and content-box.

  • background-blend-mode:
    Specifies how the background image should blend with the content. 
    background-blend-mode: multiply;
    Values include various blend modes like multiply, screen, overlay, etc.

You can use the shorthand background property to combine several of these values into a single declaration. For example:

background: #f0f0f0 url('background.jpg') center/cover no-repeat fixed;

This sets the background color, image, position, size, repetition, and attachment in one line. Adjust the values based on your design requirements.

The Border Property in CSS

In CSS, borders are used to define the edges of an element. The border property is a shorthand property that combines various individual border-related properties. These properties allow you to control the style, width, and color of an element's borders. Here are the key components of the border property:

  • border-width:
    Specifies the width of the border. You can set it to a specific value (e.g., 2px), use keywords (thin, medium, thick), or use a combination of values for different sides (top, right, bottom, left). 
    border-width: 2px;
    You can also set individual widths for each side: 
    border-width: 2px 4px 3px 1px; /* top right bottom left */
  • border-style:
    Defines the style of the border. Common values include solid, dashed, dotted, double, groove, ridge, inset, and outset. 
    border-style: solid;
    You can set different styles for each side as well: 
    border-style: solid dashed dotted double; /* top right bottom left */
  • border-color:
    Sets the color of the border. You can use color names, hex codes, RGB values, or other valid color representations. 
    border-color: #333;
    Like width and style, you can specify individual colors for each side: 
    border-color: red green blue yellow; /* top right bottom left */
  • border-radius:
    Adds rounded corners to the element. It accepts one to four values, indicating the radius of the corners. 
    border-radius: 10px;
    For individual corners: 
    border-radius: 10px 5px 15px 20px; /* top-left top-right bottom-right bottom-left */
  • border:
    The shorthand property that combines border-width, border-style, and border-color. You can use it to set all three properties in one declaration. 
    border: 2px solid #333;
    Additionally, you can include border-radius in the border shorthand: 
    border: 2px solid #333;
    border-radius: 10px;

These properties allow you to customize the appearance of borders on HTML elements, providing flexibility in creating various visual styles. Adjust these values according to your design requirements.

The Padding Property in CSS

In CSS, the padding property is used to set the space between the content of an element and its border. Padding creates an internal space within an element, and it can be applied to various HTML elements such as divs, paragraphs, and more. The padding property is a shorthand property that combines individual properties for each side of the element: top, right, bottom, and left.

Here's the basic syntax for the padding property:

padding: top right bottom left;
  • top: Specifies the padding for the top of the element.
  • right: Specifies the padding for the right side of the element.
  • bottom: Specifies the padding for the bottom of the element.
  • left: Specifies the padding for the left side of the element.

You can specify values for each side individually, or you can use fewer values to apply the same padding on multiple sides. Here are some examples:

/* Same padding on all sides */
    padding: 10px;
    
    /* Vertical padding (top and bottom) is 10px, horizontal padding (left and right) is 20px */
    padding: 10px 20px;
    
    /* Top padding is 5px, right padding is 10px, bottom padding is 15px, left padding is 20px */
    padding: 5px 10px 15px 20px;

It's important to note that the padding property adds space inside the element, pushing the content away from the element's borders. It does not affect the element's border or margin.

The Margin Property in CSS

In CSS, the margin property is used to control the space outside an element, affecting its positioning in relation to other elements. Margins create space around the borders of an element, helping control the layout and spacing between elements on a webpage. The margin property is a shorthand property that combines individual properties for each side of the element: top, right, bottom, and left.

Here's the basic syntax for the margin property:

margin: top right bottom left;
  • top: Specifies the margin for the top of the element.
  • right: Specifies the margin for the right side of the element.
  • bottom: Specifies the margin for the bottom of the element.
  • left: Specifies the margin for the left side of the element.

You can specify values for each side individually, or you can use fewer values to apply the same margin on multiple sides. Here are some examples:

/* Same margin on all sides */
margin: 10px;

/* Vertical margin (top and bottom) is 10px, horizontal margin (left and right) is 20px */
margin: 10px 20px;

/* Top margin is 5px, right margin is 10px, bottom margin is 15px, left margin is 20px */
margin: 5px 10px 15px 20px;

Margins are used to control the spacing between elements, and they do not have a background color or border. The margin space is outside the element's border. If you set a background color for an element, the margin space will not have that color.

The :hover Pseudo-class in CSS

In CSS, the :hover pseudo-class is used to apply styles to an element when a user hovers over it with the mouse cursor. This allows you to create interactive and visually appealing effects on your webpage by changing the appearance of elements in response to user actions.

Here's the basic syntax for using the :hover pseudo-class:

selector:hover {
      /* styles to apply when the element is hovered */
    }

Where selector is the HTML element or class/id selector for which you want to define hover styles.

Examples:

Changing Color on Hover:

button:hover {
      background-color: #4CAF50; /* Green color */
      color: white;
    }

Scaling on Hover:

img:hover {
      transform: scale(1.1);
    }

Underlining Links on Hover:

a:hover {
      text-decoration: underline;
    }

Displaying Hidden Content on Hover:

.hidden-content {
      display: none;
    }
    
    .parent:hover .hidden-content {
      display: block;
    }

The :hover pseudo-class is not limited to specific types of styles; it can be used to modify various CSS properties, including colors, sizes, positions, and more. It's a powerful tool for creating interactive and dynamic user experiences on your website.

Styling Links with CSS

CSS itself doesn't directly handle the creation or definition of links in the same way it handles styles for elements like paragraphs or headings. However, CSS is often used to style links to make them visually distinctive. Here's how links can be affected using CSS:

    /* Unvisited link */

    a:visited {

    a:hover {

    }

    a:active {
      color: orange;

Removing Underlines:


    /* Remove underlines from all links */
    a {
      text-decoration: none;
    }
    
    /* Underline on hover */
    a:hover {

Styling Links in Navigation Menus:


    /* Style for navigation menu links */
    .nav-link {
      padding: 10px;
      text-decoration: none;
      color: #333;
    }
    
    /* Highlight the current page */

    }

These are just a few examples of how CSS can be used to style links on a webpage. The actual structure and creation of links are handled in HTML using the <a> (anchor) element. CSS is then used to enhance the visual presentation and user experience of these links.

Float Property in CSS

The float property in CSS is used to specify how an element should be positioned within its containing element, allowing it to be moved to the left or right, and the surrounding content will flow around it. The float property is commonly used for creating layouts where elements are arranged side by side.

Float Values:


    .left-float {
      float: left;
    }
    
    .right-float {
      float: right;

Effects on Layout:


    .main-content {
      width: 70%;
      float: left;
    }
    
    .sidebar {
      width: 30%;
      float: left;

Clearing Floats:


    .clearfix::after {
      content: "";
      display: table;
      clear: both;

Apply the clearfix class to a container that contains floated elements to ensure proper clearing.

Issues and Alternatives:


    .container {
      display: flex;
      justify-content: space-between;
    }

In this example, the .container class uses flexbox to create a layout with elements evenly spaced.

It's worth noting that the float property was initially introduced for layout purposes but is now often used for non-layout purposes, such as text wrapping around images. For complex layouts, consider using newer layout techniques like Flexbox or Grid, as they provide more powerful and predictable ways to structure your page.

Position Property in CSS

In CSS, the position property is used to control the positioning of an element within its containing element. The position property can take several values, each defining a different positioning method. Here are the main values for the position property:

Static Position:


    .static-example {
      position: static;
    }

Relative Position:


    .relative-example {
      position: relative;
      top: 10px;
      left: 20px;
    }

Absolute Position:


    .absolute-example {
      position: absolute;
      top: 50px;
      left: 100px;
    }

Fixed Position:


    .fixed-example {
      position: fixed;
      top: 0;
      right: 0;
    }

Sticky Position:


    .sticky-example {
      position: sticky;

These positioning values can be used to create various layout designs and effects. It's important to understand how each value works and how they interact with other layout properties. Additionally, keep in mind that relative, absolute, fixed, and sticky positioning creates a new stacking context, which can affect how elements are layered on top of each other.

z-index Property in CSS

In CSS, the z-index property is used to control the stacking order of positioned elements along the z-axis (the axis perpendicular to the screen). The z-index property determines the order in which elements are layered on top of each other when they overlap. Elements with a higher z-index value are positioned in front of elements with a lower z-index value.

Default Stacking Order:


    .default-order {
      /* Elements are initially stacked in the order they appear in the HTML document */
    }

z-index Values:


    .element1 {
      z-index: 2;
    }
    
    .element2 {
      z-index: 1;
    }

In this example, element1 will be positioned in front of element2 because it has a higher z-index.

Stacking Context Hierarchy:


    .stacking-context {
      position: relative;
      /* Creating a stacking context */
      z-index: 1;
    }
    
    .higher-z-index {
      position: absolute;
      z-index: 2;
    }

Negative Values:


    .behind {
      z-index: -1;
    }

In this case, the element with a z-index of -1 will be positioned behind elements with higher or zero z-index values.

It's important to use the z-index property carefully, as applying large values indiscriminately can lead to confusing and difficult-to-maintain code. It's often best to use z-index sparingly and strategically, especially in complex layouts where layering is essential.

Display Property in CSS

The display property in CSS is used to control the layout behavior of an element. It defines how an element should be rendered in the document flow, affecting its visibility, box type, and interaction with other elements. The display property can take various values, each defining a different layout behavior. Here are some common values for the display property:

Display: block


    .block-element {
      display: block;
    }

Display: inline


    .inline-element {
      display: inline;
    }

Display: inline-block


    .inline-block-element {
      display: inline-block;
    }

Display: flex


    .flex-container {
      display: flex;
    }

Display: grid


    .grid-container {
      display: grid;
    }

Display: none


    .hidden-element {
      display: none;
    }

Display: table, table-row, table-cell


    .table-container {
      display: table;
    }
    
    .table-row {
      display: table-row;
    }
    
    .table-cell {
      display: table-cell;
    }

These are just a few examples of the display property values. The choice of the display value depends on the desired layout and the relationships between elements in a design. It's common to use a combination of these values to create complex layouts and achieve specific design goals.

Flexbox Layout in CSS

Flexbox, or the Flexible Box Layout, is a layout model in CSS designed to create complex layouts more efficiently and with less code. The Flexbox layout model provides an easy and predictable way to distribute space and align items within a container, even when the size of the items is unknown or dynamic.

Flex Container:


.flex-container {
  display: flex;
}

Flex Direction:


.flex-container-row {
  flex-direction: row; /* or column, row-reverse, column-reverse */
}

Flex Items:


.flex-item {
  /* Styles for flex items */
}

Flex Wrap:


.flex-container-wrap {
  flex-wrap: wrap; /* or nowrap, wrap-reverse */
}

Justify Content:


.flex-container-justify {
  justify-content: space-between; /* or flex-start, flex-end, center, space-around, space-evenly */
}

Align Items:


.flex-container-align {
  align-items: center; /* or flex-start, flex-end, stretch */
}

Align Self:


.flex-item-align-self {
  align-self: flex-end; /* or flex-start, center, stretch */
}

Flex Property:


.flex-item-flex {
  flex: 1 1 auto; /* flex-grow, flex-shrink, flex-basis */
}

Flexbox is widely used for creating responsive layouts and aligning items in a more dynamic and efficient way compared to traditional layout models. It simplifies the process of designing complex interfaces and is particularly useful when dealing with variable content or unknown dimensions.

CSS Grid Layout

CSS Grid Layout, often referred to as Grid, is a powerful two-dimensional layout system that allows you to create complex grid-based layouts with ease. It provides a set of properties and values that enable precise control over the positioning and sizing of elements in both rows and columns.

Grid Container:


    .grid-container {
      display: grid;
    }

Grid Template Columns and Rows:


    .grid-container-template {
      grid-template-columns: 100px 200px 1fr;
      grid-template-rows: auto 50% 2fr;
    }

Grid Gap:


    .grid-container-gap {
      grid-gap: 10px;
    }

Grid Areas:


    .grid-container-areas {
      grid-template-areas:
        "header header header"
        "main main sidebar"
        "footer footer footer";
    }

Grid Items Placement:


    .grid-item {
      grid-column: 2 / 4;
      grid-row: 1 / 3;
    }

Auto Placement:


    .grid-container-auto-flow {
      grid-auto-flow: column;
    }

Responsive Layouts:


    @media screen and (max-width: 600px) {
      .grid-container-responsive {
        grid-template-columns: 1fr;
      }
    }

CSS Grid Layout is widely used for designing modern and responsive web layouts. It provides a high level of control and flexibility, making it a powerful tool for building sophisticated designs.

CSS Media Queries

CSS Media Queries are a feature that allows you to apply different styles to your website based on the characteristics of the device or browser displaying the content. Media queries are often used for creating responsive designs that adapt to various screen sizes, resolutions, and orientations.

Width Range Media Query:

Orientation-based Media Query:

Styles for portrait orientation

Retina Display Media Query:

Styles for high-resolution displays
